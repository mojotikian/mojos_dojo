title:Paul's 5/5 Godroll Compendium
description:True 5/5 rolls only, one weapon at a time, current-season versions first. Four tags: PVP (before-the-kill perks), ELEM (col 3+4 elemental synergy, any element), ABTY (col 3+4 ability loops), PVE-DMG (non-elemental damage). Cols 1-2 additive perks only, chosen to compensate the weapon's weaknesses, then range/stability, then handling. Built with Claude as workflow partner, July 2026. Rebuilt 2026-07-02 on S29 Monument of Triumph hashes.

// ============================================================
// SELECTION RULES (reference for future sessions)
// 1. PVP tag: col 3+4 perks must function before a kill (no on-kill procs).
// 2. ELEM tag: col 3+4 lean into elemental subclass synergy, ANY element
//    (Voltshot, Jolting Feedback, Destabilizing Rounds, Repulsor Brace,
//    Incandescent, Hatchling, Headstone, etc). Skip if pool has none.
//    Scope widened from arc/void-only to all elements 2026-07-02, owner call.
// 2a. Within ELEM, prefer col 3+4 pairs that chain with EACH OTHER
//    (Destabilizing->Repulsor, Headstone+Firefly crystal shatter,
//    Dragonfly+Voltshot). Cross-element pairs allowed when the mechanics
//    chain. If no chaining pair exists, maximize elemental lean in both
//    columns among what remains. Added 2026-07-02, owner call.
// 3. PVE-DMG tag: remaining damage-focused traits, non-elemental.
// 4. Cols 1-2: additive only - no perk with a stat decrease. Exception:
//    fusions/bows, where lower charge time / draw time is the good direction.
// 5. Cols 1-2 priority: patch the weapon's weak stats first, then range or
//    stability, then handling, then whatever else lags.
// 6. Base perk hashes only - DIM matches enhanced versions automatically.
// 7. Masterworks can't be wishlisted; recommended MW lives in the notes line.
// 8. CURRENT SEASON FIRST: always check light.gg "Alternate Versions" and use
//    the newest item hash. Reissues get new hashes AND new perk pools - old
//    hashes silently miss current drops. When the season issues a base +
//    Upgraded pair, wishlist BOTH hashes. Kept old-season rolls get their own
//    sections later, clearly marked with their season.
// 9. ABTY tag: col 3+4 must form an ability-economy loop TOGETHER - a
//    generator or enabler (Demolitionist, Pugilist, Grave Robber,
//    Wellspring, Proximity Power) feeding a payoff (Adrenaline Junkie,
//    Golden Tricorn, Swashbuckler, One-Two Punch, Binary Orbit).
//    No two-column pair = no ABTY section, same as ELEM. Ability perks
//    may still appear on other lists when they're the best overall pick.
//    Added 2026-07-02, owner call.
// ============================================================
